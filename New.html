Stronger, Faster


"After Days" by Jack Bartman

The new UI-Engine is Stronger than ever! With the newly optimized backend for Windows
this truly marks a new era for Bit Software Co!
The new engine will be built in to the following products:
BitUI
BitUI Beta
BitUI Open Source

The following platforms will be able to run the new UI-Engine:
Windows 10
MacOS(.NET, A way to run .exe files)
Linux(.NET, A way to run .exe files)

But it hasn't been easy...

For us this new path is like a learning experience.
We can't just port over old java code and call it a
day. We need to rework the ins and outs. Due to C++
being a little anoying to work with we've had to make
some changes to the backend. These changes wont be
noticeable to average user. But with the help of the
wonder full C++ community and the help of cplusplus.com
we have been able to push away the difficulties
allowing for us to work on whats important, the user
experience.
