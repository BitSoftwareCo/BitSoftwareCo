Include the User.

The user should always have a say in what their software looks like. From the background color to the location of nav-bars. The users opinion on how they want things is very important.

What's not their Matters.

Don't be afraid of using whitespace. When used correctly it can and will look amazing.

Be bold.

Be BOLD! Use BIG text and abstract colors(such as our website). Don't be tied down by light and dark mode.

Push the User.

Push the user in the right direction of how you vison your UI but don't force it. A good example of how this can be done is by setting the default settings to be your vision, but still give the user the ability to change it.
