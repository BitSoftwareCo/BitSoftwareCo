Bevels

.bevel - Creates a Bevel Pane.

.bevel-concave - Creates a Concave Bevel Pane.

.bevel-convex - Creates a Convex Bevel Pane.

.bevel-inset - Creates a Inset Bevel Pane.

Location

.left - Aligns an element to the left.
.right - Aligns an element to the right.
.center - Aligns an element to the center.
