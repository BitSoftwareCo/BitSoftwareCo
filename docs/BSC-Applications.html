Standard Keyboard Shortcuts

What are Standard Keyboard Shortcuts? - Standard Keyboard Shortcuts are shortcuts for common actions that don't change. Actions such as Save, Open, and Copy are some Standard Keyboard Shortcuts. Here is the full list of Standard Keyboard Shortcuts:

  • Save - ctrl + s
  • Open - ctrl + o
  • Copy - ctrl + c
  • Paste - ctrl + v
  • New - ctrl + n
  • Remove - delete
  • Help - F1

Install

Inorder to install BSC Applications you'll need to enable your PC for sideloading.

Once you've enabled your PC for sideloading we can install BSC Applications!
  1. Extract "BSC.Applications.zip" - Right click "BSC.Applications.zip" and click "Extract"
  2. Open the "BSC Applications" folder
  3. Run the "Add-AddDevPackage" - Right click "Add-AddDevPackage" and click "Run with Powershell"
  4. Run "BSC Applications.appxbundle" and click "Install" / "Reinstall"

Update

Updating BSC Applications is very simple. Follow the following steps to update BSC Applications to the current version:
  1. Go to Settings
  2. Click "Check For Updates"
  3. Follow the install guide

Notes

Save and Open Notes - Notes can open and save both Rich Text and Plain Text. To open and save either you can: Click "Save" / "Open", or use the Standard Keyboard Shortcut for saving and opening files.

Formating - Notes allows you to edit and read Rich Text. To apply Rich Text to your notes you can:
  • Bold - ctrl + b
  • Italic - ctrl + i
  • Underline - ctrl + u
Note! BSC Applications removes any formated text when saving a note as Plain Text.

Phone Book

Adding Phone Numbers - To add a Phone Number enter a Name and Number and click "Add".
Note! Once you've added a Phone Number you can't remove it.

Save and Open Phone Book - Similar to notes, you can save and open phone books by: Clicking "Save" / "Open", or use the Standard Keyboard Shortcut for saving and opening files.

Photo View

Open Photos - Just like other apps, you can open photos by: Clicking "Open" or use the Standard Keyborad Shortcut for opening files.

Copy Photos - Photo View allows you to copy your opend photo and paste it somewhere else. To copy a Photo you can: Click "Copy" or Standard Keyborad Shortcut for copying objects.

Todo

Creating a New Task - Much like Phone Book, Todo allows you to create and save tasks. To create a task enter the task and click "Add".

Saving and Opening Todo Lists - To open or save a Todo List: Click "Open" / "Save" or use the Standard Keyboard Shortcut for opening and saving files.

Removing Tasks - To remove a task click "Remove" or press "Delete".
