Install

Before we start make sure you enabled your PC for sideloading(Learn how to here).

  1. Extract the "BSC Applications.zip" file.
  2. Open the "BSC Applications" folder.
  3. Right Click "Add-AddDevPackage".
  4. Click "Run With Powershell"(If there is an Error retry this step).
  5. Click "BSC Applications.appxbundle".
  6. Click "Install".
If there was any issues doing this email us or submit an issue.

Update

0.1.1+

  1. Go to Settings
  2. Click "Check For Updates"
  3. Follow the install guide.

Before 0.1.1

  1. Download BSC Applications
  2. Follow the install guide.

Notes

Open, Save, and New

  • To open a document click "Open".
  • To save a document click "Save".
  • To make a new document click "New".

Formatting

For all text formatting select the text then right click to bring up the formatting menu.
  • Bold - Bring up the formatting menu and click "B".
  • Italic - Bring up the formatting menu and click "I".
  • Underline - Bring up the formatting menu and click "U".
