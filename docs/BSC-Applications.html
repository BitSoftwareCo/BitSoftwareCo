Install

Inorder to install BSC Applications you need to be running Windows 10 1903 or later, and enable sideloading for your PC.

Once you've enabled your PC for sideloading we can install BSC Applications!

  1. Extract "BSC.Applications.zip" - Right click "BSC.Applications.zip" and click "Extract"
  2. Open the "BSC Applications" folder
  3. Run the "Add-AddDevPackage" - Right click "Add-AddDevPackage" and click "Run with Powershell"
  4. Run "BSC Applications.appxbundle" and click "Install" / "Reinstall"

Update

Updating BSC Applications is very simple. Follow the following steps to update BSC Applications to the current version:

  1. Go to Settings
  2. Go to "About"
  3. Click "Check For Updates"
  4. Follow the install guide
  5. Import your settings(1.1.0+) - After you have updated go to "Settings" > Import Settings. Go to the location where you saved your "BSC Applications Settings.json" file.

Standard Keyboard Shortcuts

What are Standard Keyboard Shortcuts? - Standard Keyboard Shortcuts are shortcuts for common actions that don't change. Actions such as Save, Open, and Copy are some Standard Keyboard Shortcuts. Here is the full list of Standard Keyboard Shortcuts:

  • Save - ctrl + s
  • Open - ctrl + o
  • Copy - ctrl + c
  • Paste - ctrl + v
  • New - ctrl + n
  • Remove - delete

Temporary Content

What is Temporary Content? - Temporary Content is a form of storage used when navigating between apps. When you go to navigate from app to app BSC Applications stores all of your content, so that when you come back you can start where you left off.

  • Disable - To disable Temporary Content go to Settings > Temporary Content. When you flip the switch to off it will clear and stop saving your Temporary Content.
  • Clear - To clear your Temporary Content go to Settings > Clear. Please note that this action cannot be undone.
